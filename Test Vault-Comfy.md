## Extension FF: MarkDownload

Installe MarkDownload sur Firefox (addons.mozilla.org).[](https://github.com/deathau/markdownload)​  
Configure : Options → "Folder inside Downloads/ to store clips" = nom_dossier (ex. "Inbox"), active Obsidian integration (vault: Vault-Comfy, folder: Inbox) + plugin Advanced URI dans Obsidian.  
Sélectionne texte/fil IA → clic droit → "MarkDownload: Markdown from selection" → .md prêt ; rafraîchis vault.[](https://addons.mozilla.org/en-US/firefox/addon/obsidianmark/)
